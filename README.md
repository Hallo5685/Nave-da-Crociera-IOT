Progetto NaveIOT

Questo progetto scolastico prevede la simulazione della misurazione di temperatura e umidità in una o più navi da crociera, con più ponti e cabine.

Allo scopo di compilare il codice dell'ultima versione:
-scaricare la repository
-posizionarsi nella cartella "NaveIOT/DA" 
-lanciare il comando "python3 iotgwda.py"
-aprire un altro terminale e possizionarsi in "NaveIOT/DC"
-lanciare il comando "python3 dc.py"
-per interrompere la visualizzazione, di entrambi gli scrypt, lanciare il comando "Ctrl+C" nei rispettivi terminali aperti
