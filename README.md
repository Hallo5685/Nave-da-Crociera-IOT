Progetto NaveIOT

Questo progetto scolastico prevede la simulazione della misurazione di temperatura e umidità in una o più navi da crociera, con più ponti e cabine.

Allo scopo di compilare il codice nella versione del 22/01/2026 12:45 bisogna:
-scaricare la repository
-posizionarsi nella cartella "NaveIOT" 
-lanciare il comando "python3 -m DC.dc"
-per interrompere la visualizzazione lanciare "Ctrl+C"
